# Nike Sales Data Analysis

This project performs **data cleaning and exploratory data analysis (EDA)** on a Nike sales dataset using Python.

The goal of the project is to transform raw sales data into a clean dataset and analyze patterns in product sales.

---

## Project Overview

Real-world datasets are often messy and require cleaning before analysis.
In this project, the raw Nike sales dataset is cleaned and explored to understand sales trends and product performance.

The project demonstrates:

* Data cleaning
* Handling missing values
* Exploratory Data Analysis (EDA)
* Data preprocessing

---

## Dataset

The dataset used in this project:

**Nike_Sales_Uncleaned.csv**

It contains raw sales information such as product details, sales numbers, and other attributes.

After cleaning the dataset, the processed data is saved as:

**nike_sales_cleaned.csv**
